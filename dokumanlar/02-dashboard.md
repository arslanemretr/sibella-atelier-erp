# Dashboard

Sayfa:

- `/dashboard`

Amaci:

- Kullaniciya genel ERP ozeti gostermek
- Ana menulere gecisi hizlandirmak

Calisma sekli:

- Dashboard acildiginda ozet kutulari ve hareket listeleri gosterilir.
- Sayfa genel yonlendirme ve durum takibi icindir.
- Hareket ekranlarindan gelen yeni kayitlar ileride bu alana yansitilabilir.

Not:

- Dashboard bu surumde agir raporlama yerine yonetsel ozet gorevi gorur.
